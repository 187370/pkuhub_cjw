关于PKUHUB

平台简介

PKUHUB是一个由北大学生自主开发的学习资源共享网站，旨在促进校内知识流通，提高学习效率。平台汇集了各院系、各课程的往年试题、学习笔记、课件资料等，方便同学们查阅和分享。

平台秉持"知识共享、互助共进"的理念，鼓励同学们在合法合规的前提下，分享自己的学习资料，同时充分尊重知识产权，共同构建良好的学习生态。

平台规则

  • 上传的资料必须是用户自己创作或拥有分享权利的内容
  • 严禁上传侵犯他人知识产权的内容
  • 不得上传与学习无关的内容或违反法律法规的内容
  • 用户应对自己上传的内容负责
  • 平台保留对违规内容进行删除的权利

技术实现

本平台基于 Python Flask 框架开发，采用 SQLite 数据库存储用户和资料信息，前端使用 Tailwind CSS 构建响应式界面。
